# MrappsAmazonBundle

##Full Configuration##

```
mrapps_amazon:
    parameters:
        access: YOUR_AMAZON_ACCESS_KEY_HERE
        secret: YOUR_AMAZON_SECRET_KEY_HERE
        region: YOUR_AMAZON_REGION_HERE
        default_bucket: DEFAULT_BUCKET
    cdn:
        enable: true/false
        url: YOUR_CDN_URL_HERE
```
